CollegeStudentTools: This is a flask website which assist college students with tools.
	More specifically, the website includes a Student Loan Calculator and a Expense Tracker.

Programs:
	driver.py (main program)
		(Main Program. driver.py will startup the localhost web application.)
	
Installations Required: 
	-Python	3+
	-Flask
		-pip install Flask
Tools Used:
	-Python (Flask)
	-HTML
	-CSS
	-Bootstrap
	-Javascript

Instructions:
1. Run driver.py. This will start the web application that will run on localhost.

4. In your browser, enter 127.0.0.1:5000 as the url.
